How sync works
IB Flex (Web): pulls trades, positions, dividends, and cashflows via IB Flex Web Service.
Chase / RJ (Offline): ingests exported CSV/PDF statements placed in a data directory or uploaded per-connection.
Coverage is COMPLETE when the latest run is successful and parsing had no failures; otherwise PARTIAL.
