Limitations

  • The basket is the current Campaign 2 30-name basket, so the 2006-2025 test is survivorship-biased and does not include delisted names or period-correct sector leaders.
  • The panel is availability-aware: a ticker can only enter after valid adjusted-price history exists; no replacement ticker is substituted for missing early history.
  • Years before every basket constituent has valid history may have fewer active names, so equal-weight basket concentration can be higher than in full-coverage years.
  • Adjusted prices include split/dividend effects where the upstream provider supplies them; provider revisions can change historical adjusted series.
  • The campaign is research-only and intentionally does not change live trading defaults, broker routing, or agent production policy.
  • NOT survivorship-free / static-theme proxy
  • Theme membership is a static sub-basket over current campaign constituents, so survivorship and hindsight bias can inflate results.
  • The run can disprove TCS cheaply, but it cannot certify edge without point-in-time thematic membership and delisted-name coverage.
  • Tax-aware oversize trims use offline loss/LTCG-aware lot ordering; production tax-lot routing is not invoked because this is an offline research backtest.
  • Single-name harvesting is limited to rule exits: oversize trims and momentum-decay loss relegation.
